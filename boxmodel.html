What is Css box model

This a box that wraps around every Html element
It also consists of the content,margins,borders and padding

Properties of box model

  • content-Where text and images appears
  • Padding-Clears an area around the padding
  • Border-A border that goes around the padding amd content
  • Margin-Clears an area outside the border

How to calculate the total dimension of a box

+400px*2:Width
+20px*2:padding
+2px*2:Border
+20px*2;Margin
Total=444px
